--2023.1.16创建

写这个的目的就是在linux A上 搭建一个 类似于ftp的服务器, 在linux B上可以随意上传和下载文件
实现主要功能
mkdir 在服务器上创建文件夹
ls    查看文件夹
服务器文件名 可以用ls 的序列号做替代 cd命令会清除序列号

push -f A  [b] 将 文件A push到文件服务器 [b](文件名 不写就是默认) -f强制推送覆盖
pull -f A  [b] 同理 服务器 A拉去下来   A可以是文件序列号(有限匹配文件名)
rm   -rf 删除服务器文件类linux 

ls返回值
序列号 文件名 (全目录)


消息使用protocbuf
网络和消息回调使用boost库asio
